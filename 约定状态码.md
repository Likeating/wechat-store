
**返回参数说明**

```
  "meta": {
    "msg": "登录成功",
    "status": 200
  }
```
| 参数名 | 类型   | 说明                |
| :----- | :----- | ------------------- |
| msg    | string | 错误信息说明        |
| status | number | 状态码              |

status：100-500 为 HTTP状态码
status>=600为约定状态码

601 token无效，请重新登录。
602 token过期，请重新登录。
603 token算法不一致！
604 用户未注册。
